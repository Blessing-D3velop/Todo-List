# To-Do List App

A simple and interactive **To-Do List** web application built with **HTML, CSS, and JavaScript**.  
This project helps users organize tasks, and delete tasks in a user-friendly interface.

##  Features

- Add new tasks dynamically
- Delete tasks
- Responsive design for desktop and mobile
- Persistent task list (if using local storage in your code)

##  Technologies Used

- **HTML5** – structure of the app  
- **CSS3** – styling and layout, responsive design  
- **JavaScript** – task management functionality
